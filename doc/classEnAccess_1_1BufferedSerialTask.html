Detailed Description

Turns BufferedSerial into a Task. Normally, performReadWrite() would be called from an interrupt handler as soon as new data is available from the serial hardware. On platforms where an interrupt is not available and the serial hardware needs to be polled instead (like Unix termios), this class can be used to do the polling in a Task and and it to the Scheduler.
